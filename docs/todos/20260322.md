Tích hợp ggwave — Wave Signin

Xem doc chi tiết: docs/thoughts/wave-qr-signin-architecture.md

---

NHIỆM VỤ:
Hoàn thiện hệ thống signin offline-first qua sóng âm ggwave, song song với passkey.

ỨNG DỤNG:
- Thiết bị A (đã đăng nhập) truyền seed cho thiết bị B (chưa đăng nhập) qua loa/micro.
- Không cần server, không cần mạng. QR đã loại khỏi flow này — chỉ dùng âm thanh.

THIẾT KẾ ĐÃ CHỐT (xem doc):
- Hai vai trò tách biệt: bên xin (ui-authenticate) và bên cho (ui-authorize)
- Wire format ký tự đơn: "~" epub, ":" type, "!" ct, "@" iv, "#" s, "&" chunk index
- Bên cho tự listen khi mount, không cần thao tác user
- Bên xin bắt đầu bằng nút "Request"

CẦN LÀM:

[x] 1. Refactor ui-authenticate (src/UI/components/authenticate/)
    - State machine mới: neutral → requesting → listening
    - Neutral: hiện epub rút gọn (abcde...vwxyz) + nút "Request"
    - Requesting: sea.pair() → waveEl.listen() → phát { "~": epub, ":": ">" } → chuyển listening
    - Listening: xử lý message đến
      - Nhận { ":": "!>" } → hiện "Access denied"
      - Nhận { "~", "!", "@", "#" } → sea.secret + sea.decrypt → Access.wave({ seed }) → emit "done"
    - Nút Stop → dừng waveEl → về neutral
    - File: authenticate/wave.js (WaveAuth class), authenticate/index.js

[x] 2. Tạo ui-authorize component (src/UI/components/authorize/) — folder hiện đang rỗng
    - State machine: listening → confirming → listening
    - connectedCallback: tự gọi waveEl.listen()
    - Confirming khi nhận { "~": epub_B, ":": ">" }:
      - Hiện: "Thiết bị abcde...vwxyz xin cấp quyền truy cập"
      - Nút [Deny] → phát { ":": "!>" } → về listening
      - Nút [Grant] → sea.secret(epub_B, pair_A) → sea.encrypt(seed, secret, {raw:true})
                    → phát { "~": pair_A.epub, "!": ct, "@": iv, "#": s } → về listening
    - Files cần tạo: index.js, template.js, styles.css.js

[x] 3. Refactor route /authorize (src/UI/routes/authorize/)
    - template.js hiện render <ui-qr data-mode="scan" /> → đổi thành <ui-authorize />
    - index.js giữ thin, không chứa protocol logic
    - Đảm bảo import ui-authorize

[x] 4. Refactor chunking trong ui-wave (src/UI/components/wave/index.js)
    - Format cũ: { __waveChunk, id, index, total, part } → xóa
    - Format mới: { "&": 0, ":": part } cho chunk giữa; { "&": "N!", ":": part } cho chunk cuối
    - Receiver: detect key "&" → tích lũy → khi typeof parsed["&"] === "string" && endsWith("!") → join
    - Cập nhật maxsize nếu cần (overhead key đã giảm)

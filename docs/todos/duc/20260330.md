## Vấn đề ban đầu

Khi build hash trong Node.js và browser cho torrent, kết quả khác nhau → không tái tạo được BitTorrent protocol → server và browser đứng ở 2 swarm khác nhau → không tìm thấy nhau dù cùng hold một file.


```javascript
// Bên trong bundle:
b = window || self          // Node.js: cả hai = undefined
subtle = b.crypto.subtle    // → subtle = undefined → THROW
```

Node.js không có `window` hay `self` → bundle throw error → không tính được InfoHash.

---

## Solution đã implement

**File**: `src/core/Torrent/hashInfo.js`

Dùng `create-torrent` **direct import** (không qua bundle). Package `uint8-util` tự route đúng:
- Node.js → `_node.js` → `crypto.createHash('sha1')`
- Browser → `browser.js` → `subtle.digest('SHA-1')`

Cả hai SHA-1 → **output identical** cho cùng input.

```javascript
import createTorrent from "create-torrent"
import bencode from "bencode"
import { hash } from "uint8-util"

export async function hashInfo(data, name, opts = {}) {
    const buf = await new Promise((resolve, reject) =>
        createTorrent(data, { name, pieceLength: opts.pieceLength },
            (err, b) => err ? reject(err) : resolve(b))
    )
    const { info } = bencode.decode(buf)
    return hash(bencode.encode(info), "hex")
}
```

## Cách dùng

**Node.js — build time (pre-compute InfoHash):**
```javascript
import { hashInfo } from "../src/core/Torrent/hashInfo.js"
import { readFile, writeFile } from "fs/promises"

const buffer = await readFile("./build/statics/db/app.sqlite")
const infoHash = await hashInfo(buffer, "app.sqlite")
await writeFile("./build/statics/db/app.sqlite.torrent-hash", infoHash)
```

**Browser — verify + join swarm:**
```javascript
// Browser KHÔNG gọi hashInfo() trực tiếp
// Lấy InfoHash từ server pre-computed file:
const serverHash = await fetch("/statics/db/app.sqlite.torrent-hash").then(r => r.text())

// Join swarm của server:
const torrent = await t.add(`magnet:?xt=urn:btih:${serverHash}`)

// Sau khi download xong, re-seed:
const torrent2 = await t.seed(file)
// torrent2.infoHash === serverHash ✓
```

## Rule bắt buộc

| Rule | Lý do |
|---|---|
| `name` PHẢI explicit | Không pass → `"Unnamed Torrent {Date.now()}"` → non-deterministic mỗi lần gọi |
| `name` PHẢI identical Node.js & Browser | Khác tên → khác InfoHash → khác swarm |
| Dùng `hashInfo()` trong Node.js | Browser dùng `webtorrent.seed(file).infoHash` |

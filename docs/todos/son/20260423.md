# Task: Add Object Type Support to ZEN PEN VM

## Mục tiêu tổng quát

Nâng cấp ZEN PEN VM (`pen.zig` + `pen.js`) để hỗ trợ kiểu dữ liệu object (JSON object). Hiện tại khi pass một object vào register của PEN, nó bị silent cast thành `null` (tag 0) trước khi vào WASM. Cần thêm: wire tag mới, opcode `ISO` (is-object) và `JGET` (JSON field get), cập nhật bytecode builder, viết unit test, build, test, publish lên npm, rồi `npm update` trong akao.

## Kiến trúc hiện tại (PHẢI nắm rõ trước khi sửa)

### Luồng dữ liệu khi gọi `pen.run(bytecode, regs)`

1. **pen.js** nhận `regs` array từ JS (các giá trị: string, number, null, bool, object)
2. `_writeReg()` trong pen.js serialize từng register vào linear memory của WASM với wire format:
   - tag 0 = null/undefined
   - tag 1 = bool (1 byte: 0/1)
   - tag 2 = integer i64 LE (8 bytes)
   - tag 3 = f64 LE (8 bytes)
   - tag 4 = string (2 bytes LE length + utf8 bytes)
   - **Hiện tại**: object → tag 0 (null) ← CẦN SỬA
3. **wasm.zig** `parseRegs()` đọc wire format từ buffer → tạo `pen.Value` array
4. **pen.zig** `run()` / `eval()` thực thi bytecode với các `Value` đó

### pen.zig internal Value tags
```zig
const TAG_NULL:  u8 = 0;
const TAG_BOOL:  u8 = 1;
const TAG_INT:   u8 = 2;
const TAG_FLOAT: u8 = 3;
const TAG_STR:   u8 = 4;
const TAG_ERR:   u8 = 5;
// TAG_OBJ = 6  ← CẦN THÊM
```

### Opcode ISA hiện tại (liên quan)
```
0x60 ISS(a)   — is string?
0x61 ISN(a)   — is number?
0x62 ISX(a)   — is null?
0x63 ISB(a)   — is bool?
0x64 LNG(a,min,max) — string length in range
0x65          — FREE ← dùng cho ISO
0x80 SEGR(reg,sep,idx)
0x81 SEGRN(reg,sep,idx)
0x82          — FREE ← dùng cho JGET
```

### 1. Thêm `TAG_OBJ = 6` vào `src/pen.zig`

```zig
const TAG_OBJ: u8 = 6;
```

Thêm function `vObj`:
```zig
// vObj — store an object by packing (pointer, length) into Value.i.
// KHÔNG copy bytes vào s[]. JSON bytes ở nguyên trong buf của wasm.zig.
// WASM32: pointer = u32, length = u32; packed vào i: i64 dưới dạng u64.
pub fn vObj(ptr: [*]const u8, len: usize) Value {
    var v: Value = undefined;
    v.tag = TAG_OBJ;
    const p32: u32 = @intCast(@intFromPtr(ptr));
    const l32: u32 = @intCast(if (len > 0xffffffff) @as(usize, 0xffffffff) else len);
    v.i = @bitCast(@as(u64, p32) | (@as(u64, l32) << 32));
    v.f = 0;
    v.slen = 0; // unused for TAG_OBJ
    return v;
}

// Helper: extract pointer from TAG_OBJ Value
fn objPtr(v: Value) [*]const u8 {
    const u: u64 = @bitCast(v.i);
    return @ptrFromInt(@as(u32, @truncate(u)));
}

// Helper: extract length from TAG_OBJ Value
fn objLen(v: Value) usize {
    const u: u64 = @bitCast(v.i);
    return @as(u32, @truncate(u >> 32));
}
```

Cập nhật `isTruthy`: TAG_OBJ → length > 0:
```zig
TAG_OBJ => objLen(v) > 0,
```

### 3. Thêm opcode `0x82 JGET(obj, key)` vào `src/pen.zig`

JGET nhận 2 child expressions: obj_expr (phải có tag TAG_OBJ) và key_expr (phải có tag TAG_STR).  
Trả về giá trị của field đó từ JSON object được store ở s[0..slen].

Thêm vào `skipExpr` (2 children, binary):
```zig
0x30, 0x31, 0x32, 0x33, 0x34, 0x35,
0x40, 0x41, 0x42, 0x43, 0x44, 0x45,
0x55, 0x56, 0x57, 0x58, 0x59, 0x82 => {
    try skipExpr(bc, pos, depth - 1);
    try skipExpr(bc, pos, depth - 1);
},
```

Thêm vào `eval`:
```zig
0x82 => { // JGET(obj_expr, key_expr) — extract field from JSON object
    const obj = try eval(ctx, pos);
    const key = try eval(ctx, pos);
    if (obj.tag != TAG_OBJ) return vNull();
    if (key.tag != TAG_STR) return vNull();
    // Reconstruct JSON slice from packed pointer+length (no copy)
    const json = objPtr(obj)[0..objLen(obj)];
    return jsonGet(json, key.s[0..key.slen]);
},
```

Thêm các helper functions và `jsonGet` vào `src/pen.zig`:

```zig
// skipJsonWs — bỏ qua whitespace
fn skipJsonWs(json: []const u8, i_: usize) usize {
    var i = i_;
    while (i < json.len and (json[i] == ' ' or json[i] == '\t' or json[i] == '\n' or json[i] == '\r')) : (i += 1) {}
    return i;
}

// skipJsonString — bỏ qua một JSON string, i_ trỏ vào ký tự NGAY SAU dấu `"` mở.
// Xử lý đúng backslash escapes (\" \\, v.v.).
// Trả về position NGAY SAU dấu `"` đóng.
fn skipJsonString(json: []const u8, i_: usize) usize {
    var i = i_;
    while (i < json.len) {
        if (json[i] == '\\') { i += 2; continue; } // bỏ qua cặp escape
        if (json[i] == '"') return i + 1;           // đóng string
        i += 1;
    }
    return i;
}

// skipJsonValue — bỏ qua một JSON value bất kỳ (string, number, bool, null, object, array).
// i_ trỏ vào ký tự đầu tiên của value (sau khi skip whitespace).
fn skipJsonValue(json: []const u8, i_: usize) usize {
    var i = skipJsonWs(json, i_);
    if (i >= json.len) return i;
    switch (json[i]) {
        '"' => return skipJsonString(json, i + 1),
        '{', '[' => {
            const open = json[i];
            const close: u8 = if (open == '{') '}' else ']';
            i += 1;
            var depth: u32 = 1;
            while (i < json.len and depth > 0) {
                if (json[i] == '"') { i = skipJsonString(json, i + 1); continue; }
                if (json[i] == open) depth += 1;
                if (json[i] == close) { depth -= 1; }
                i += 1;
            }
            return i;
        },
        else => { // number, bool, null
            while (i < json.len) : (i += 1) {
                const c = json[i];
                if (c == ',' or c == '}' or c == ']' or c == ' ' or c == '\t' or c == '\n' or c == '\r') break;
            }
            return i;
        },
    }
}

// readJsonStringValue — đọc nội dung string (i_ sau dấu `"` mở), unescape, ghi vào Value.s.
// Trả về TAG_STR Value. String bị truncate ở MAX_STR bytes.
fn readJsonStringValue(json: []const u8, i_: usize) Value {
    var v: Value = undefined;
    v.tag = TAG_STR;
    v.i = 0; v.f = 0;
    var out: usize = 0;
    var i = i_;
    while (i < json.len and out < MAX_STR) {
        if (json[i] == '\\' and i + 1 < json.len) {
            i += 1;
            v.s[out] = switch (json[i]) {
                '"'  => '"',
                '\\' => '\\',
                '/'  => '/',
                'n'  => '\n',
                'r'  => '\r',
                't'  => '\t',
                'b'  => 8,
                'f'  => 12,
                else => json[i],
            };
            out += 1;
            i += 1;
            continue;
        }
        if (json[i] == '"') break;
        v.s[out] = json[i];
        out += 1;
        i += 1;
    }
    v.slen = @intCast(out);
    return v;
}

// parseJsonNumber — parse number (int or float) starting at i_.
fn parseJsonNumber(json: []const u8, i_: usize) Value {
    var i = i_;
    var is_float = false;
    if (i < json.len and json[i] == '-') i += 1;
    while (i < json.len and json[i] >= '0' and json[i] <= '9') : (i += 1) {}
    if (i < json.len and json[i] == '.') {
        is_float = true; i += 1;
        while (i < json.len and json[i] >= '0' and json[i] <= '9') : (i += 1) {}
    }
    if (i < json.len and (json[i] == 'e' or json[i] == 'E')) {
        is_float = true; i += 1;
        if (i < json.len and (json[i] == '+' or json[i] == '-')) i += 1;
        while (i < json.len and json[i] >= '0' and json[i] <= '9') : (i += 1) {}
    }
    const fval = parseF64(json[i_..i]);
    if (!is_float) {
        const ival: i64 = @intFromFloat(fval);
        if (@as(f64, @floatFromInt(ival)) == fval) return vInt(ival);
    }
    return vFloat(fval);
}

// parseJsonValue — parse một JSON value tại vị trí i_, trả về Value.
fn parseJsonValue(json: []const u8, i_: usize) Value {
    const i = skipJsonWs(json, i_);
    if (i >= json.len) return vNull();
    const c = json[i];
    if (c == '"') return readJsonStringValue(json, i + 1);
    if (c == 'n' and i + 4 <= json.len and json[i+1]=='u' and json[i+2]=='l' and json[i+3]=='l') return vNull();
    if (c == 't' and i + 4 <= json.len and json[i+1]=='r' and json[i+2]=='u' and json[i+3]=='e') return vBool(true);
    if (c == 'f' and i + 5 <= json.len and json[i+1]=='a' and json[i+2]=='l' and json[i+3]=='s' and json[i+4]=='e') return vBool(false);
    if ((c >= '0' and c <= '9') or c == '-') return parseJsonNumber(json, i);
    if (c == '{' or c == '[') {
        // nested: return TAG_OBJ trỏ vào đúng đoạn bytes trong buf
        const end = skipJsonValue(json, i);
        return vObj(json[i..].ptr, end - i);
    }
    return vNull();
}

// jsonGet — tìm field theo key trong JSON object, trả về Value.
// Hỗ trợ đầy đủ: escaped quotes trong keys và values, object/array lồng nhau,
// JSON dài tuỳ ý (không truncate input — chỉ string VALUE bị truncate ở MAX_STR).
fn jsonGet(json: []const u8, key: []const u8) Value {
    var i = skipJsonWs(json, 0);
    if (i >= json.len or json[i] != '{') return vNull();
    i += 1; // skip '{'

    while (true) {
        i = skipJsonWs(json, i);
        if (i >= json.len or json[i] == '}') return vNull();
        if (json[i] == ',') { i += 1; continue; }
        if (json[i] != '"') return vNull(); // malformed
        i += 1; // skip opening '"'

        // Scan this key — hỗ trợ escape sequences trong key
        const key_start = i;
        while (i < json.len) {
            if (json[i] == '\\') { i += 2; continue; }
            if (json[i] == '"') break;
            i += 1;
        }
        const this_key = json[key_start..i];
        if (i < json.len) i += 1; // skip closing '"'

        i = skipJsonWs(json, i);
        if (i >= json.len or json[i] != ':') return vNull();
        i += 1;
        i = skipJsonWs(json, i);

        // So sánh key (không có escapes trong key vì keys luôn là plain identifiers)
        const matches = (this_key.len == key.len) and blk: {
            for (this_key, key) |a, b| if (a != b) break :blk false;
            break :blk true;
        };

        if (matches) return parseJsonValue(json, i);

        // Bỏ qua value
        i = skipJsonValue(json, i);
    }
}
```

### 4. Cập nhật `src/wasm.zig` — thêm wire tag 5 cho object

Trong `parseRegs()`, thêm case trước `else`:

```zig
5 => { // object: JSON encoded as 2 bytes LE length + utf8
    if (off + 2 > buf.len) {
        regs[i] = pen.vNull();
        break;
    }
    const slen: u16 = @as(u16, buf[off]) | (@as(u16, buf[off + 1]) << 8);
    off += 2;
    if (off + slen > buf.len) {
        regs[i] = pen.vNull();
        break;
    }
    regs[i] = pen.vObj(buf[off..].ptr, slen);
    off += slen;
},
```

### 5. Cập nhật `src/pen.js` — wire encoder + bytecode builder

#### 5a. `_writeReg` — serialize object thành wire tag 5

Thay dòng `view[offset++] = 0; return offset;` cuối function (hiện tại là fallback cho object) bằng:

```javascript
// object/array: JSON-encode, write as tag 5
if (typeof val === "object") {
  var json;
  try { json = JSON.stringify(val); } catch(e) { json = "{}"; }
  view[offset++] = 5;
  var encoded = _enc.encode(json);
  var slen = Math.min(encoded.length, 0xffff);
  view[offset++] = slen & 0xff;
  view[offset++] = (slen >> 8) & 0xff;
  for (var i = 0; i < slen; i++) view[offset++] = encoded[i];
  return offset;
}
view[offset++] = 0;
return offset;
```

**Vị trí chèn**: sau block `if (typeof val === "string") { ... }`, trước dòng `view[offset++] = 0; return offset;`.

#### 5c. `treeskip` — thêm cases cho 0x65 và 0x82

Hiện tại:

Đổi thành:
```javascript

```

Và thêm case cho `0x82` (2 children, binary):
```javascript

  pos = treeskip(bytecode, pos);

  pos = treeskip(bytecode, pos);

  return treeskip(bytecode, pos);

  pos = treeskip(bytecode, pos);
```

```javascript
// ── PEN object support ────────────────────────────────────────────────────────

describe("PEN object support", function () {
  it("bc.iso() opcode exists", function () {
    assert.ok(typeof pen.bc.iso === "function");
    var bytes = pen.bc.iso(pen.bc.r1());
    assert.strictEqual(bytes[0], 0x65);
  });

  it("bc.jget() opcode exists", function () {
    assert.ok(typeof pen.bc.jget === "function");
    var bytes = pen.bc.jget(pen.bc.r1(), pen.bc.str("side"));
    assert.strictEqual(bytes[0], 0x82);
  });

  it("ISO(R[1]) accepts plain object", function () {
    var bc = pen.bc;
    var bytecode = prog(bc.iso(bc.r1()));
    assert.strictEqual(run(bytecode, ["k", { side: "buy" }]), true);
  });

  it("ISO(R[1]) rejects string", function () {
    var bc = pen.bc;
    assert.strictEqual(run(prog(bc.iso(bc.r1())), ["k", "hello"]), false);
  });

  it("ISO(R[1]) rejects number", function () {
    var bc = pen.bc;
    assert.strictEqual(run(prog(bc.iso(bc.r1())), ["k", 42]), false);
  });

  it("ISO(R[1]) rejects null", function () {
    var bc = pen.bc;
    assert.strictEqual(run(prog(bc.iso(bc.r1())), ["k", null]), false);
  });

  it("ISO(R[1]) accepts empty object (truthy check)", function () {
    var bc = pen.bc;
    assert.strictEqual(run(prog(bc.iso(bc.r1())), ["k", {}]), true);
  });

  it("JGET extracts string field", function () {
    var bc = pen.bc;
    var bytecode = prog(bc.eq(bc.jget(bc.r1(), bc.str("side")), bc.str("buy")));
    assert.strictEqual(run(bytecode, ["k", { side: "buy" }]), true);
    assert.strictEqual(run(bytecode, ["k", { side: "sell" }]), false);
  });

  it("JGET extracts number field", function () {
    var bc = pen.bc;
    var bytecode = prog(bc.eq(bc.jget(bc.r1(), bc.str("qty")), bc.uint(10)));
    assert.strictEqual(run(bytecode, ["k", { qty: 10 }]), true);
    assert.strictEqual(run(bytecode, ["k", { qty: 5 }]), false);
  });

  it("JGET extracts bool field", function () {
    var bc = pen.bc;
    var bytecode = prog(bc.jget(bc.r1(), bc.str("active")));
    assert.strictEqual(run(bytecode, ["k", { active: true }]), true);
    assert.strictEqual(run(bytecode, ["k", { active: false }]), false);
  });

  it("JGET returns null for missing field", function () {
    var bc = pen.bc;
    var bytecode = prog(bc.isx(bc.jget(bc.r1(), bc.str("missing"))));
    assert.strictEqual(run(bytecode, ["k", { side: "buy" }]), true);
  });

  it("JGET on non-object returns null", function () {
    var bc = pen.bc;
    var bytecode = prog(bc.isx(bc.jget(bc.r1(), bc.str("side"))));
    assert.strictEqual(run(bytecode, ["k", "not_an_object"]), true);
  });

  it("{ type: 'object' } in ZEN.pen() spec accepts object", function () {
    var soul = ZEN.pen({ val: { type: "object" }, open: true });
    var bc = pen.unpack(soul.slice(1));
    assert.strictEqual(pen.run(bc, ["k", { a: 1 }, soul, 0, Date.now(), ""]), true);
    assert.strictEqual(pen.run(bc, ["k", "string", soul, 0, Date.now(), ""]), false);
    assert.strictEqual(pen.run(bc, ["k", 42, soul, 0, Date.now(), ""]), false);
  });

  it("{ jget: [...] } in ZEN.pen() spec validates field", function () {
    var soul = ZEN.pen({
      val: {
        and: [
          { type: "object" },
          { jget: ["side", { or: [{ eq: "buy" }, { eq: "sell" }] }] },
        ],
      },
      open: true,
    });
    var bc = pen.unpack(soul.slice(1));
    var regs = function (v) { return ["k", v, soul, 0, Date.now(), ""]; };
    assert.strictEqual(pen.run(bc, regs({ side: "buy", qty: 1 })), true, "buy accepted");
    assert.strictEqual(pen.run(bc, regs({ side: "sell", qty: 2 })), true, "sell accepted");
    assert.strictEqual(pen.run(bc, regs({ side: "hold" })), false, "hold rejected");
    assert.strictEqual(pen.run(bc, regs("not_object")), false, "string rejected");
  });

  it("JGET nested: second-level field (object value of a field)", function () {
    var bc = pen.bc;
    // val = { base: { type: "NFT" } }
    // JGET(R[1], "base") → TAG_OBJ
    // ISO(JGET(R[1], "base")) → true
    var bytecode = prog(bc.iso(bc.jget(bc.r1(), bc.str("base"))));
    assert.strictEqual(run(bytecode, ["k", { base: { type: "NFT" }, qty: 1 }]), true);
    assert.strictEqual(run(bytecode, ["k", { base: "string_value" }]), false);
  });

  it("object register round-trips through wire encoder without corruption", function () {
    var bc = pen.bc;
    var obj = { side: "buy", qty: 5, active: true };
    var bytecode = prog(
      bc.and([
        bc.iso(bc.r1()),
        bc.eq(bc.jget(bc.r1(), bc.str("side")), bc.str("buy")),
        bc.eq(bc.jget(bc.r1(), bc.str("qty")), bc.uint(5)),
        bc.jget(bc.r1(), bc.str("active")),
      ])
    );
    assert.strictEqual(run(bytecode, ["k", obj]), true);
  });

  it("scanpolicy is not affected by ISO/JGET bytes in expression tree", function () {
    var soul = ZEN.pen({ val: { type: "object" }, sign: true });
    var bc = pen.unpack(soul.slice(1));
    var p = pen.scanpolicy(bc);
    assert.strictEqual(p.sign, true, "sign correctly detected with ISO in tree");
  });

  it("ISO object at graph level: open:true soul accepts object write", function (done) {
    this.timeout(8000);
    var zen = Zen({ radisk: false, peers: [], localStorage: false });
    var soul = ZEN.pen({ val: { type: "object" }, open: true });
    zen.get(soul).get("k").put({ side: "buy" }, function (ack) {
      assert.ok(!ack || !ack.err, "object write must be accepted: " + (ack && ack.err));
      done();
    });
  });

  it("ISO: string rejected at graph level for object-type soul", function (done) {
    this.timeout(8000);
    var zen = Zen({ radisk: false, peers: [], localStorage: false });
    var soul = ZEN.pen({ val: { type: "object" }, open: true });
    zen.get(soul).get("k").put("string_value", function (ack) {
      assert.ok(ack && ack.err, "string must be rejected by object-type soul: " + JSON.stringify(ack));
      done();
    });
  });
});
```

## Constraints quan trọng

1. **Không break backward compatibility**: objects passé cũ (→ null) bây giờ trở thành TAG_OBJ. Các bytecode cũ không có ISO/JGET vẫn chạy đúng. Tuy nhiên nếu ai đã pass object trước đây và expect null, behavior thay đổi. Đây là acceptable breaking change vì passing objects trước đây là undefined behavior.

2. **Input JSON KHÔNG bị truncate**: `vObj` lưu pointer + length trực tiếp vào `Value.i` (packed u64), không copy vào `s[128]`. JSON trong `buf` được scan in-place. Giới hạn duy nhất là 64KB total của `buf` (wire format dùng u16 length = max 65535 bytes JSON per register).

3. **String VALUE trả về từ JGET bị truncate ở MAX_STR=128**: `readJsonStringValue` ghi vào `v.s[0..MAX_STR]`. Điều này OK vì PEN VM chỉ dùng string values để compare (EQ, PRE, SUF, REGEX) — nếu value > 128 bytes thì compare sẽ fail, là behavior hợp lý.

4. **Escaped quotes trong keys được handle**: `skipJsonString` xử lý đúng `\"` nên key như `"fo\"o"` được skip đúng. Keys thực tế là plain ASCII nên không cần match escaped keys.

5. **Wire tag 5 = object trong wasm.zig** ≠ **TAG_OBJ = 6 trong pen.zig**. Đây là intentional: wire tag và internal tag là 2 không gian riêng biệt.

6. **WASM linear memory stability**: `vObj` lưu raw pointer vào `buf` của wasm.zig. Pointer này valid suốt duration của `run()` call vì `buf` là static global array không bị move hay free. Safe.

5. **Zig phải available** để compile pen.wasm. Nếu không có zig, dùng pre-built. Trong trường hợp này, tạm thời không thể compile WASM mới, cần tìm cách khác (xem phần "Nếu zig không available" bên dưới).

## Nếu zig không available trong môi trường

Builder (`lib/builder/pen.js`) tự detect: nếu `zig` không có hoặc bị block, nó sẽ log `"zig not available, skipping pen.wasm build (using pre-built binary)"` và không fail. Khi đó:

- `src/pen.zig` và `src/wasm.zig` được sửa nhưng `pen.wasm` không được recompile
- Tests liên quan đến object support sẽ fail vì WASM binary cũ không có TAG_OBJ
- **Giải pháp**: cần tìm môi trường có zig để compile, hoặc cross-compile từ CI

Kiểm tra zig availability: `zig version` — nếu exit code ≠ 0, zig không available.

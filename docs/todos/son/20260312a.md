## 1. File mới: `src/UI/components/identicons/`

### `index.js` — class `IDENTICONS`

**Attributes nhận từ parent (qua dataset), đọc qua getters:**
| Attribute | Getter | Mô tả | Ví dụ |
|---|---|---|---|
| `data-seed` | — | `observedAttributes` watch thay đổi → re-render toàn bộ | `"0x1a2b…"` |
| `data-name` | `get name()` | tên radio group | `"avatar"`, `"wallet"` |
| `data-size` | `get size()` | kích thước identicon con | `"7"`, `"5"` |
| `data-step` | `get step()` | step mỗi lần +/- (fallback `5`) | `"5"` |

```javascript
get name() { return this.dataset.name || "item" }
get size() { return Number(this.dataset.size) || 5 }
get step() { return Number(this.dataset.step) || 5 }
```

**Seed flow (2 tầng hash):**
```
Access.get("seed")               ← raw seed, nằm trong JS memory, KHÔNG ra DOM
    ↓ sea.work(seed, "avatar")   ← avatars hash 1 lần với namespace
data-seed của ui-identicons      ← hash 1 chiều, an toàn trên DOM attribute
    ↓ sea.work(this.dataset.seed, id)  ← identicons hash thêm 1 lần với index
data-seed của từng ui-identicon  ← hash 2 chiều, chỉ dùng để vẽ SVG pattern
```

> `avatars` và `wallets` tự gọi `sea.work()` rồi set `this.$identicons.dataset.seed = hashedSeed`.  
> Khi `data-seed` thay đổi (`attributeChangedCallback`): clear container → re-render toàn bộ.

**JS setters (parent cập nhật động):**
| Property | Mô tả |
|---|---|
| `set id(value)` | lưu vào `$id`, cập nhật radio đang checked |
| `set total(value)` | lưu vào `$total`, gọi `render()` |

**Events (child → parent communication):**

Dùng `new Events()` — sau khi nâng cấp `Events.js`, mỗi instance có `$events` riêng biệt, không còn là singleton.  
`identicons` tạo `this.events = new Events()` trong constructor, parent lắng nghe qua `this.$identicons.events.on(...)`, cleanup theo pattern `subscriptions` đã có trong codebase.

```javascript
// identicons
this.events = new Events()
this.events.emit("select", { id })
this.events.emit("increase")
this.events.emit("decrease")

// avatars / wallets
this.subscriptions.push(
    this.$identicons.events.on("select", ({ detail: { id } }) => { ... }),
    this.$identicons.events.on("increase", () => { ... }),
    this.$identicons.events.on("decrease", () => { ... })
)
```

> `events.on()` đã trả về unsubscribe function — push thẳng vào `subscriptions[]` là đủ, không cần `removeEventListener` thủ công.

| Event | `detail` | Khi nào |
|---|---|---|
| `"select"` | `{ id: Number }` | user click 1 identicon |
| `"increase"` | — | user click nút `+` |
| `"decrease"` | — | user click nút `-` |

**Methods nội bộ:**
- `async create()` — sinh seed: `sea.work(this.dataset.seed, id)` → `data-seed` cho từng `<ui-identicon>`; sau khi xong enforce checked state từ `_id`
- `remove()` — xóa item thừa từ cuối
- `clear()` — xóa toàn bộ container + reset `_subs` (xem Bug 2)
- `render()` — debounce qua `queueMicrotask`, gọi `create()`/`remove()`; chỉ chạy nếu `$total > 0` và `dataset.seed` tồn tại (xem Bug 4)

**Bug fixes tích hợp vào thiết kế:**

> **Bug 2 — memory leak click listeners:** Mỗi lần `create()` chạy, nó push cleanup của label click listener vào một array riêng `this.$subs = []` (không phải `this.subscriptions`). Khi `clear()` gọi, `$subs` được flush: `this.$subs.forEach(off => off()); this.$subs = []`. `this.subscriptions` chỉ chứa button listeners của `#increase`/`#decrease`.

> **Bug 3 — identicons thiếu disconnectedCallback:** `connectedCallback` gắn listener cho `#increase`/`#decrease` và push cleanup vào `this.subscriptions`. `disconnectedCallback` chạy `this.subscriptions.forEach(off => off())`.

> **Bug 4 — double render khi initial load:** `attributeChangedCallback` (khi `data-seed` đổi) gọi `clear()` rồi gọi `render()` — nhưng `render()` guard: `if (!this.$total || !this.dataset.seed) return`. Parent phải gọi `set total` SAU khi đã set `dataset.seed` để trigger render thực sự chỉ một lần.

> **Bug 5 — `set id` silent fail trước khi DOM tồn tại:** Cuối `create()`, sau khi render xong, luôn enforce: `const input = this.$container.querySelector(\`input#i${this.$id}\`); if (input) input.checked = true`.

**Import:** `"/UI/components/identicon/index.js"` — tái sử dụng, không viết lại logic SVG

## 2. Refactor `src/UI/components/avatars/`

### `index.js` — những thay đổi

**Xóa:** `create()`, `remove()`, `select()`, `increase()`, `decrease()`, `render()`

```javascript
connectedCallback() {
    this.$identicons = this.shadowRoot.querySelector("ui-identicons")

    const seed = async () => {
        const hashed = await globalThis.sea.work(Access.get("seed"), "avatar")
        this.$identicons.dataset.seed = hashed
    }

    this.subscriptions.push(
        this.$identicons.events.on("select", ({ detail: { id } }) => { this.id = id }),
        this.$identicons.events.on("increase", () => { this.total += this.step }),
        this.$identicons.events.on("decrease", () => { if (this.total - this.step > this.id) this.total -= this.step }),
        Access.on("authenticated", async ({ value }) => {
            this.style.display = value ? "flex" : "none"
            if (value) { await seed(); this.render() }
            else this.$identicons.clear()
        }),
        Access.on("avatar", () => this.render())
    )
    if (Access.get("authenticated")) seed().then(() => this.render())
}

render() {
    this.$identicons.id = this.id
    this.$identicons.total = this.total
}
```

> `sync()` bị loại — thay bằng `render()` (single-word). Không cần `sync()` riêng vì avatars không có `change()`.

**Giữ nguyên:** `get id`, `set id`, `get total`, `set total` (domain logic `setAvatar`)

### `template.js`

```html
<ui-identicons data-name="avatar" data-size="7">
    <ui-context data-key="dictionary.avatars" />
</ui-identicons>
```

### `styles.css.js`

Chỉ còn `:host { display: flex; flex-direction: column; gap: var(--space) }`

## 3. Refactor `src/UI/components/wallets/`

### `index.js` — những thay đổi

**Xóa:** `create()`, `remove()`, `select()`, `increase()`, `decrease()`, `render()`

```javascript
// Trong connectedCallback
this.$identicons = this.shadowRoot.querySelector("ui-identicons")

const seed = async () => {
    const hashed = await globalThis.sea.work(Access.get("seed"), "wallet")
    this.$identicons.dataset.seed = hashed
}

this.subscriptions.push(
    this.$identicons.events.on("select", ({ detail: { id } }) => { this.id = id }),
    this.$identicons.events.on("increase", () => { this.total += this.step }),
    this.$identicons.events.on("decrease", () => { if (this.total - this.step > this.id) this.total -= this.step }),
    Access.on("authenticated", async ({ value }) => {
        this.toggleAttribute("hidden", !value)
        if (value) { await seed(); this.change() }
        else {
            this.states.set({ address: null })
            this.$identicons.clear()
        }
    }),
    Access.on("wallet", this.change)
    // states.on("chain") và states.on("currency") bị xóa — change() đã xử lý đủ
)

if (Access.get("authenticated")) seed().then(() => this.change())
```

> **`sync()` bị loại hoàn toàn** — `change()` đã có sẵn và làm mọi thứ.  
> `this.states.on("chain", this.render)` và `this.states.on("currency", this.render)` cũng bị xóa vì chain/currency thay đổi luôn đi qua `change()`.

**`change()`** — thay `this.render()` bằng 2 dòng update identicons trực tiếp:
```javascript
async change(event) {
    if (!Access.get("authenticated")) return
    if (event?.target) {
        // ... xử lý SELECT như cũ
    }
    // Replaces this.render()
    this.$identicons.id = this.id
    this.$identicons.total = this.total
    if (this.states.get("chain") && this.states.get("currency")) {
        // ... update balance/address như cũ
    }
}
```

> **Bug 1 fix:** `seed()` được gọi trong `Access.on("authenticated")` và lúc initial load.

Thời đại của offline-first in-browser server đã tới

Stacks:
- GunDB: Real-time graph database sync
- WebTorrent: in-browser torrent, biến browser thành nguồn cung dữ liệu
- WebRTC: kết nối thẳng giữa các browser, không cần server tập trung
- OPFS: lưu file như FS, ngay trên trình duyệt, biến trình duyệt thành host
- SQLite WASM: truy xuất dữ liệu phức tạp tốc độ bàn thờ
- IndexedDB: cache json, svg, human-readable files

Hiện nay hệ thống đã có GDB (gun), IDB (cache), SQL (bọc SQLite WASM), OPFS. Cần tích hợp thêm:

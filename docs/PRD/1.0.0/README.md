## Quy ước

- `(ref: ...)` — Reference đến source code hoặc docs làm evidence
- `[TBD]` — Cần confirm, có lý do cụ thể
- `[ASSUMPTION]` — Giả định, cần validate
- `[DECISION NEEDED]` — Cần quyết định từ stakeholder
- `[CONFLICT]` — Logic mâu thuẫn giữa các source
- ✅ — Đã implement trong code
- ⚠️ — Implement một phần
- 🔲 — Chưa implement (requirement cần build)
